# Sales Data Analysis Report

## Project Overview

This project analyzes sales data using Python and Pandas to extract meaningful business insights.

## Objectives

- Load and explore real-world data
- Clean missing values
- Calculate key sales metrics
- Generate a simple analytical report

## Tools Used

- Python
- Pandas
- VS Code

## Dataset Description

- Rows: 10
- Columns: 5
- File: sales_data.csv

## Data Cleaning

- Filled missing Quantity values with 0
- Calculated missing Total_Sales
- Removed duplicate records

## Analysis Performed

- Total Revenue
- Average Sales Value
- Highest and Lowest Sale
- Best Selling Product

## Key Findings

- Best selling product identified using groupby
- Revenue metrics calculated using Pandas functions

## Conclusion

This project demonstrates basic data analysis, data cleaning, and reporting using Pandas.
